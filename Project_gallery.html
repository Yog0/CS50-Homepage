FSAE is a global competition where students compete to design and build a Formula 1-style open wheel race car. In 2019 I joined the RMIT team as the Electronics Lead. With the help of an awesome team, I designed, built and coded the electrical and electronics system. It was an incredible experience, and the team had its best run since 2008. We actually managed to break the event acceleration record!

The Calculator is a simple tool I built for The Odin Project. It...well...works like a calculator. Use the buttons to enter numbers and perform operations. This was a good little challenge, as it involved a lot of work with HTML, CSS and Javascript that really pushed me to the boundaries of what I could do at the time. Check it out here.

Etch-A-Sketch was another project for The Odin Project. Mouse over the grid to draw an image with random colors. As you hover over squares, they will get darker and darker before eventually turning black. The grid can be initialised for any size up to 64 by 64. This project was a great CSS learning experience, as it forced me to figure out how to fit a variable amount of content (the user-entered grid size) into the fixed size of the panel. You can see it here.
